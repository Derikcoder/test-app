1.5 Consumables
Afrikaans: Verbruikbare Voorraad
Definition:
- Sum of all sundry expenses excluding consumables multiplied by the business-defined sliding scale.
- This should be treated as a derived value, not a directly procured stock line, unless the invoice explicitly itemizes it that way.

Recommended default sliding-scale rule:
- Use the total of sundry expenses excluding consumables as the consumables base.
- Apply one bracket only, based on the final base amount.
- As cost increases, the percentage decreases to prevent high-value components from inflating consumables disproportionately.

Consumables base:
- consumablesBase = sum(Service Parts and Spares + Components + Third Party Service Providers + Lubricants and Concentrates)

Recommended brackets:
- R0.00 to R1,000.00 => 7.5%
- R1,000.01 to R2,500.00 => 5.0%
- R2,500.01 to R5,000.00 => 3.5%
- Above R5,000.00 => 2.5%

Boundary handling:
- Exactly R1,000.00 falls into the 7.5% bracket.
- Exactly R2,500.00 falls into the 5.0% bracket.
- Exactly R5,000.00 falls into the 3.5% bracket.
- Any amount greater than R5,000.00 uses 2.5%.

Suggested formula:
- consumablesCost = consumablesBase * applicableBracketRate

Worked examples:
- If consumablesBase = R850.00, consumablesCost = R850.00 * 0.075 = R63.75
- If consumablesBase = R1,800.00, consumablesCost = R1,800.00 * 0.05 = R90.00
- If consumablesBase = R4,200.00, consumablesCost = R4,200.00 * 0.035 = R147.00
- If consumablesBase = R12,000.00, consumablesCost = R12,000.00 * 0.025 = R300.00

Rationale:
- Lower-value jobs tend to consume proportionally more shop materials relative to parts cost.
- Higher-value jobs may include expensive components that should not distort consumables pricing.
- A decreasing bracket model keeps consumables commercially reasonable and easier to defend with customers.

2) FIXED EXPENSES
Afrikaans: Vaste Uitgawes
Definition: Fixed unit-cost defined items total.

Subsections:
- Call-out fee
- Procurement Fee
Afrikaans: Verkryging Koste
- Labor

Suggested formula:
- fixedExpenses = callOutFee + procurementFee + laborCost

3) TOTALS

3.1 Total (VAT Excl)
Afrikaans: Totaal / BTW Uitgesluit
Formula:
- totalExclVat = sundryExpenses + fixedExpenses

3.2 VAT
Afrikaans: BTW
Rule:
- vatRate = 15%
- vatAmount = totalExclVat * 0.15

3.3 Total (VAT Incl)
Afrikaans: Totaal / BTW Ingesluit
Formula:
- totalInclVat = totalExclVat * 115 / 100

MAPPING NOTES FOR THE APP
- Parts line items map naturally into the existing parts/line-items model.
- Labor remains a separate cost concern.
- Call-out fee remains a fixed-expense concern.
- Consumables should remain derived from the parts-side cost base using business rules.
- Procurement Fee should be carried separately from raw parts totals so profitability remains visible.
